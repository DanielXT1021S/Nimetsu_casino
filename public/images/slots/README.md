# Imágenes de Slots

## Instrucciones para la imagen de la máquina NIMETSU

Guarda la imagen de la máquina tragamonedas en este directorio con el nombre:
- `nimetsu-slot-machine.png`

La imagen será usada como fondo decorativo de la máquina de slots.

## Tamaño recomendado
- Ancho: 700-800px
- Alto: 900-1100px
- Formato: PNG con transparencia

La imagen se centrará y se ajustará automáticamente al contenedor.
